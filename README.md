# Extracao_StackOverFlow
Técnicas de WebScraping para extração de dados relevantes.

Esse é o projeto que desenvolvemos na disciplina de Análise de Texto de Fontes Desestruturadas e Web. Nosso principal objetivo é aplicar as técnicas aprendidas em sala de aula para criar uma prova de conceito envolvendo a extração e análise de dados de fontes desestruturadas. Teremos a oportunidade de exercitar nossa autonomia na escolha da fonte de dados e na aplicação das técnicas de processamento de linguagem natural, mineração de texto e análise de sentimentos. Através desse projeto, esperamos aprimorar nossas habilidades na análise de dados não estruturados e aplicar de forma prática os conhecimentos adquiridos durante o curso. 
Nele, extraímos informações do site https://stackoverflow.com/questions, são extraídas informações diárias do site, para fazer uma análise adequada das questões que são discutidas na atualidade.
